# SPORTSPACE

Deployment-ready Streamlit app for booking university sports facilities.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Features

- Landing page and button-based navigation
- Discover Sports → Facilities → Booking flow
- End-to-end booking confirmation with generated booking IDs
- Session-state booking storage
- My Bookings table
- Community posts and post creation
- Leaderboard charts
- Profile metrics
- No database, no CSV dependency
